1. What is a Computer?
Ans : A computer is an electronic device that processes data and performs various tasks according to a set of instructions.

2. What is RAM?
Ans : RAM (Random Access Memory) is a type of computer memory that temporarily stores data and instructions that the CPU (Central Processing Unit) needs to access quickly.

3. Where is data stored in a computer?
Ans : Data in a computer is stored in various storage devices like hard drives, solid-state drives (SSDs), USB flash drives, and other types of memory devices.

4. What is the input device used to type text and numbers on a document in the computer system?
Ans : The input device used for typing text and numbers on a computer is a keyboard.

5. What are the output devices?
Ans : Output devices in a computer system include monitors (or screens), printers, speakers, and other devices that display or provide processed information to the user.

6. Which is the input device that allows a user to move the cursor or pointer on the screen?
Ans : The input device used to move the cursor or pointer on the screen is a mouse or a touchpad.

7. Which language is directly understood by the computer without a translation program?
Ans : Machine language, consisting of binary code (1s and 0s), is directly understood by the computer without needing a translation program.

8. What are Input devices?
Ans : Input devices are hardware components that allow users to enter data or commands into a computer system. Examples include keyboards, mice, touchscreens, scanners, and microphones.
